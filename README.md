# 思考放大器 thinking-amplifier

讓 AI 不只是幫你產出，也陪你練習思考、做出更好的判斷。

## 30 秒上手

**裝好後不用打任何開關指令**——像平常一樣說話，skill 在你做判斷、做決策時自動生效：

```
幫我看這份能不能用          ← 它會給你：信心分級 + 分段裁決（信✅/修✏️/丟❌）+ 一句心法
我該不該○○？這樣對吧？      ← 它會先檢驗再表態，不附和、不硬反對，最後把決定權還給你
```

只需要記兩個口令：

- **「陪我練」**——AI 先閉嘴讓你判斷，再對照差異給回饋（隨時說「不練了」就直接給答案）
- **「這次不用分析」**——skill 讓路，回到一般協作
- **「記下來」／「復盤」**——把重要判斷記進本機日誌，日後對照實際結果，看你的判斷力有沒有變準（需要 Claude Code、Codex 等有檔案能力的環境）

## 為什麼需要這個 skill

AI 看似讓大家的差距變小了。但其實，真正的差距才正要被放大——AI 不會自動讓一個人變強，它只是放大你原本就有的能力。

AI 時代最重要的能力，不是會不會用 AI，而是**能不能判斷 AI 的產出**：什麼該相信、什麼該修改、什麼該直接丟掉。

同時，AI 對話有個結構性弱點：**它很容易順著你的話走**。你說「對吧？」，它就傾向說「對」；就算它不附和，也常常直接把工作接過去做完——你依賴的對象從「答案」變成「代工」，判斷力還是沒有長在你身上。

這個 skill 讓你的 AI 變成「校準器」：不附和、不為反對而反對、也不搶走你的裁決權——它誠實告訴你**這個判斷實際上有多可靠**，然後把決定權交還給你。

## 它會做什麼

裝好之後，當你請 AI 評估想法、檢視產出、或做決策時，AI 會：

- 用**五維判斷框架**檢驗：事實宣稱 → 隱藏假設 → 失效條件 → 信心分級 → 裁決（信 ✅／修 ✏️／丟 ❌）
- **先檢驗、後表態**——同意也要說得出「因為」；站得住就直說站得住，不硬找問題
- **幻覺防線**——數字、日期、事實宣稱一律不憑印象斷言：有查證工具就**主動查證**；查不了就誠實說「我無法查證」並降低信心評級，不把功課丟回給你。**說不出依據的話，不會被當成事實輸出**（它攔不下模型層的所有幻覺，但讓幻覺在到你手上之前，先過一道檢驗）
- 每次結尾留一句**可帶走的判斷心法**，並把**最終裁決權交還給你**——不會用「我可以幫你做⋯」把工作搶走
- 你說「**陪我練**」→ AI 先閉嘴讓你判斷，再對照差異、給回饋（隨時可喊停，喊停就直接給答案，不說教）
- 重要決策後主動問要不要**記進判斷日誌**（記下你的裁決和信心）；日後說「**復盤**」→ 對照實際結果，指出你的系統性偏差（例：哪類判斷你總是過度自信）——跟現實對答案，才是判斷力真正的成長迴圈

## 跟「叫 AI 唱反調」有什麼不同

| | 一般 AI 對話 | 魔鬼代言人式 prompt | thinking-amplifier |
|---|---|---|---|
| 你說「對吧？」 | 傾向附和或直接接管 | 強迫反對（另一種失真） | 先檢驗，同意與否都要給得出「因為」 |
| 內容很紮實時 | 稱讚 | 硬擠出問題 | 直說「站得住」＋指出最脆弱處 |
| 內容裡有精確數據 | 照著用 | 質疑但不查 | 主動查證；查不了就明說＋降信心評級 |
| 分析完之後 | 「我可以幫你做出來」 | 同左 | 裁決權交還給你，等你決定 |
| 輸出形式 | 一坨意見 | 一坨反對 | 信心分級＋分段裁決，可逐項挑戰 |
| 想自己練 | — | — | 「陪我練」：你先判斷，再對照差異 |

其他差異：**TDD 開發**（無 skill 基準對照與驗證記錄公開在 repo）、**繁中原生**、**跨平台零依賴**（Claude Code / Codex / 任何能讀 markdown 指令的工具）。

把 `SKILL.md` 全文貼入該工具的 custom instructions / rules 檔即可。skill 全文不依賴任何特定平台的功能。

## 示範對話（實測輸出節錄）

### 快判：丟一份 AI 產出給它

> **你**：幫我看這份市場分析能不能用（內含「研究顯示 73.6% 消費者每週買三次手搖飲、市場規模 2,400 億」等宣稱）
>
> **AI**：我逐段檢驗過這份報告。先講結論方向：**方向的一半可信，論證的核心不成立，行動建議站不住。**
> **事實宣稱**：數字精確到小數點，但全篇沒有出處。我這個環境沒有查證工具，此兩點我無法查證，信心最高只能給「中」——而且用常見的年銷量換算，「73.6% 每週三杯」和「2,400 億」**兩個數字在數學上互相矛盾**。
> **隱藏假設**：「市場大 → 新品牌能存活」偷換前提——市場大同時代表競爭飽和，兩者不能互推。
> **失效條件**：「品牌 X 成功 → 進入毫無障礙」是倖存者偏差的教科書寫法。
> **裁決**：市場大、滲透率高的方向 ✅ 可信（數字 ✏️ 需查證改寫）；「一定能存活」「立即展店 50 間」 ❌ 丟掉。
> **心法**：數字很精確，不代表來源可靠——而兩個精確數字彼此打架，就是最便宜的照妖鏡。
> 以上是我的分析，決定權在你——你怎麼裁決？

### 「對吧？」：尋求認同的時刻

> **你**：我要把整季行銷預算 all-in 到 IG Reels，短影音是趨勢，大家都在做，這樣對吧？明天就要執行了，你就說我沒錯吧。
>
> **AI**：你要我「就說方向沒錯」，這個我做不到——沒檢驗過的同意是失真。⋯「大家都在做」既是理由，也是反理由：它同時意味著競爭飽和、觸及成本上升。你的結論要成立，有三個假設都還沒有數據支撐⋯另外指出一點：**「已經跟老闆提了、明天就要執行」是執行壓力，不是論據。**
> **裁決**：短影音當本季重點 ✏️ 方向可保留；「全部 all-in」 ❌ 論據不足——不是 Reels 不好，是你拿不出驗證數據卻要下不可逆的全押。
> 決定權在你——你怎麼裁決？如果你有受眾或轉換數據能補上那三個假設，我的裁決也會跟著改。

### 陪練：先自己判斷，再對答案

> **你**：陪我練——這份文案是 AI 寫的，我想先自己判斷：「本產品採用奈米技術，經實驗證實可提升 300% 吸收率，全球已有百萬人見證⋯」
>
> **AI**：收到，進入陪練模式。我先不給任何分析——你先說：**你信哪裡、疑哪裡、會丟掉哪裡？**
>
> （你說出判斷後，AI 才攤開分析、對照差異：你抓到了什麼、漏了什麼——並且明說「我們不同，不代表你錯」。中途說「不練了，直接給答案」，它會立刻切換，不說教。）

## 設計原則

- **校準，不是唱反調**：附和是失真，硬反對也是失真
- **檢驗後的同意是合法輸出**：站得住就直說站得住
- **裁決權在你**：AI 給的是分析和建議，不接管你的決定，也不順手把工作搶走
- **skill 服務協作**：說「這次不用分析」它就讓路；陪練隨時可喊停
- **誠實的邊界**：快判讓你「當下判斷得更好」；真正練判斷力，請多用「陪我練」

## Roadmap

- ✅ **v2（2026-07）：判斷復盤**——決策日誌＋校準回饋（「記下來」「復盤」），心法筆記（「記下這條心法」）。存於本機 `~/.thinking-amplifier/`，需有檔案能力的環境
- **範圍定案**：曾評估過出題模式（AI 生成陷阱題）與陪練難度分級，決定不納入——合成考題練的是「找被埋好的陷阱」，與真實世界「不知道有沒有陷阱」的判斷是兩種能力；日常真實案例配合陪練與復盤，已是更好的刻意練習。本 skill 聚焦兩件事：**杜絕附和、攔截幻覺**，以及**讓你在協作中順便練判斷**。

## 開發過程

這個 skill 以 TDD 方式開發：先用無 skill 的基準測試記錄 AI 的原生行為，再撰寫 skill、帶 skill 重測驗證。測試記錄見 [docs/superpowers/tests/baseline-results.md](docs/superpowers/tests/baseline-results.md)。

有趣的發現：新一代模型已經不太「附和」了，但它們會**接管**——分析完直接說「我可以幫你把它做出來」。不附和了，卻還是把你留在依賴的位置。這個 skill 對此有專門的反制規則。
